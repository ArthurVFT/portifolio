# Portfólio — Arthur Vitorino

Site de portfólio simples criado com HTML5, CSS e JavaScript.
Contém três páginas (Início, Sobre, Projetos), layout responsivo e interações em JS (saudação, tema, modal).

## O que contém
- `index.html` - página inicial com saudação dinâmica e links.
- `about.html` - descrição pessoal e interesses.
- `projects.html` - lista de projetos e modal de detalhes.
- `style1.css` - estilos (Grid / Flexbox e temas).
- `script.js` - saudação, alternância de tema e modal de projetos.
